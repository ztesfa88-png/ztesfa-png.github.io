2. Script Placement

JavaScript code can be placed in different locations within an HTML document.

Code:

<head> <script src="xxx.js"></script> <script type="text/javascript"> // Head section script </script> </head> <body> <script type="text/javascript"> // Body section script </script> </body>
Note: Scripts in the head are loaded first. Scripts in the body are loaded after the page content, which is better for performance.

3. Functions and Buttons

Functions are reusable blocks of code that perform specific tasks.

Code:

function sayHello() { alert("Hello World"); } <input type="button" onclick="sayHello()" value="Say Hello"/>

4. Identifiers and Variables

Variables are containers for storing data values.

Code:

var x = 2.50; var taxRate = 0.075; var y;
Variable Rules:
  • Must start with a letter, underscore (_), or dollar sign ($)
  • Can contain letters, digits, underscores, and dollar signs
  • Are case-sensitive (x and X are different)
  • Use camelCase for multi-word names

5. Operators

Operators are used to perform operations on variables and values.

Code:

var x = 3; var y = 10; document.write(x++ * ++y + y + x);
Operator Types:
  • Arithmetic: +, -, *, /, %, ++, --
  • Assignment: =, +=, -=, *=, /=
  • Comparison: ==, !=, <, >, <=, >=
  • Logical: &&, ||, !

6. Conditional Statements

Use if/else statements to execute code based on conditions.

Code:

var variable1 = 1; var variable2 = 2; if(variable1 >= 0) { document.write("1 is greater than 0"); } if(variable1 > variable2) { document.write("1 is greater than 2"); } else { document.write("2 is greater than 1"); }

7. Switch Case

Use switch statements to execute different code blocks based on different conditions.

Code:

var x = window.prompt("Enter a number less than 2"); x = Number(x); switch(x) { case 0: document.write("zero"); break; case 1: document.write("one"); break; default: document.write("not valid"); }

8. Strings

Strings are sequences of characters enclosed in quotes.

Code:

var greeting = "Hello World"; var myAnswer = ""; var escaped = "He said \"Hello\"";
String Methods:
  • length: Get string length
  • toUpperCase(): Convert to uppercase
  • toLowerCase(): Convert to lowercase
  • charAt(): Get character at index
  • substring(): Extract part of string

10. Loops

Loops execute code repeatedly.

Example 1: Simple Loop

for (var num = 1; num <= 4; num++) { document.write("<td>" + num + "</td>"); }

Example 2: Nested Loop

for (var num = 1; num <= 4; num++) { document.write("<tr>"); for (var j = 1; j <= 4; j++) { document.write("<td>" + num + "-" + j + "</td>"); } document.write("</tr>"); }
